# Learning Notes

## GitHub Actions
- Continuous integration and delivery (CI/CD).
- Automate workflows using `.yaml` files.
- Integrates with GitHub repositories seamlessly.

## Jenkins
- Open-source automation server.
- Supports building, deploying, and automating software projects.
- Utilizes plugins for customization.

## Docker
- Containerization platform for packaging applications.
- Allows consistent environments across development and production.
- Simplifies deployment and scaling of applications.

## Kubernetes
- Container orchestration platform.
- Manages containerized applications across a cluster.
- Handles deployment, scaling, and management of applications.

## AWS/DevOps
- Cloud services to support infrastructure and application deployment.
- Terraform for infrastructure as code.
- CI/CD pipelines integrated with AWS services like CodePipeline.

## Kafka
- Distributed event streaming platform.
- Handles real-time data feeds with high throughput.
- Utilizes producer-consumer architecture for message brokering.

## Snyk Vulnerability Check Extension
- Tool for identifying and fixing vulnerabilities in applications.
- Integrates with GitHub to automate security checks in CI/CD pipelines.
- Provides remediation guidance for vulnerable dependencies.
